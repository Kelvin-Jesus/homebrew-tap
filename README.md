# Kelvin-Jesus Tap

## How do I install these formulae?

`brew install kelvin-jesus/tap/<formula>`

Or `brew tap kelvin-jesus/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "kelvin-jesus/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
